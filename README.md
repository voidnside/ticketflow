# TicketFlow — Ticketing & SLA Management System

A system designed to simulate real-world customer support operations, including ticket management, SLA tracking, and issue categorization.

## Why this project

This project reflects real support workflows and focuses on how issues are tracked, prioritized, and resolved.

## Features

- Ticket lifecycle management
- SLA tracking (response & resolution)
- Role-based access (agent/admin)
- Issue categorization

## Tech Stack

Node.js, TypeScript, Next.js, PostgreSQL

## What this demonstrates

- API design
- System thinking
- Handling real-world support workflows
